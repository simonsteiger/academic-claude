# academic-claude

A set of Claude skills to support academic work. <img src="/docs/src/assets/logo.svg" align="right" alt="academic-claude logo" style="height: 140px;">

This skill set currently includes:

- `summarise-article`: Summarise an article
- `review-article`: Add comments on a scientific text

The structure and design of these skills is inspired by [superpowers](https://github.com/obra/superpowers).
